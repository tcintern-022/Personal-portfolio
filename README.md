# Mahad Khan — Personal Portfolio

A personal portfolio landing page built with **plain HTML and CSS only**

## About

This is the portfolio site for **Mahad Khan**, a 7th semester BSCS student at COMSATS University Islamabad, Wah Campus, and an AI Intern focused on Machine Learning, Data Engineering, and Generative AI.

## Sections

- **Navigation bar** — fixed header with a CSS-only (checkbox-based) mobile menu, no JS required
- **Hero** — intro with name, role, and an animated SVG graph background built with CSS keyframes
- **About Me** — background, education, and current focus areas
- **Skills** — grouped into Languages, AI / Generative AI, ML / Data, Databases, and Core CS
- **Projects** — two academic projects (Financial Fraud Pattern Mining, Hotel Management System) with tech stacks and descriptions
- **Contact** — direct links (email, GitHub, LinkedIn) plus a contact form

## Tech

- HTML5 (semantic elements)
- CSS3 (Flexbox layout, CSS variables, keyframe animations, responsive media queries)
- Google Fonts: Space Grotesk, Inter, JetBrains Mono
- No Bootstrap, no Tailwind, no JavaScript

## Project Structure

```
.
├── index.html    # Page structure and content
├── styles.css    # All styling, layout, and responsiveness
└── README.md
```

## Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/tcintern-022/personal-portfolio.git
   ```
2. Open the folder in VS Code (or any editor).
3. Open `index.html` directly in a browser, or use the **Live Server** VS Code extension for auto-reload while editing.
